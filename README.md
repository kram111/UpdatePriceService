# UpdatePriceService

## Перед запуском:
Исправьте данные в appsettings.json (в папке Service и Service/publish_service, а именно почту и зашифрованный пароль

## Описание публичного АРІ:
### Базовый URL
http://localhost:5054/api/
### Эндпоинты
1. Post - /subscribe
2. Get - /subscriptions
3. Delete - /unsubscribe

## Исполняемые файлы 
1. В папке UpdatePriceService/src/Client/publish_client/Client
2. В папке UpdatePriceService/src/Service/publish_service/Service

## Запуск (из корня UpdatePriceService)
1. Перейти ./src/Service/publish_service/Service, и запустить сервис
2. Перейти ./src/Client/publish_client/Client, запустить сервис и дейтсвовать по справке
